## 与后端的通讯协议（已在 request.ts 实现）

- 请求体统一 AES-CBC 加密（随机 IV 前置 + Base64），密钥与 Vue2 版本一致（`utils/crypto.ts`）。
- 响应 `data` 字段为 AES 密文，拦截器内解密后 `JSON.parse`。
- 请求头：`X-Token`（登录 token）、`X-Request-Time`（秒级时间戳）、`X-Request-Id`（uuid，防重放）。
- 业务码：`0` 成功、`-999` 鉴权失败（跳登录页）、`-3` 需强制绑定 2FA。
- 开发端口：vite dev 默认 `3002`（被占用会自动顺延），`/api` 代理到 `http://127.0.0.1:26666`。

## 与后端的关系

- 后端仓库：`C:\huawei\goproject\SamWaf`（Go），管理端口 `:26666`（HTTPS）。
- 管理端 API 走 `/api` 前缀，开发时需在 `vite.config.ts` 配置代理（Vue2 版本的代理目标为 `https://127.0.0.1`，新项目建议直接指向 `https://127.0.0.1:26666` 并设置 `secure: false`）。
- 生产构建产物（`dist/`）需拷贝到后端 `vue/` 目录后一起编译打包进二进制。
- 接口加密/认证逻辑参考 Vue2 版本 `src/utils/` 与 `src/apis/`（crypto-js 加解密、Token 处理）。
