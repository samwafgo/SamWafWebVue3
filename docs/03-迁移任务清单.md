# Vue2 → Vue3 业务页面迁移任务清单

> 依据老项目 `SamWafWeb/src/router/modules/waf.ts` 拆分，按依赖关系排序，依次迁移。
> 行数为老项目 Vue2 源码行数，用于评估工作量。
> 状态标记：✅ 完成 / 🔄 进行中 / ⬜ 未开始
> 创建日期：2026-06-10

## 迁移约定（每个页面都遵守）

1. Options API → `<script setup lang="ts">` 组合式 API
2. `this.$store`（Vuex）→ Pinia store；`this.$t` → `useI18n().t`；`this.$message` → `MessagePlugin`
3. `tdesign-vue` → `tdesign-vue-next`（注意属性差异：`visible.sync` → `v-model:visible` 等）
4. API 文件从 `src/apis/` 平迁，request 封装已兼容（加密/Token/防重放头一致）
5. 路由按组注册到 `src/router/modules/waf.ts`，meta.title 用老语言包键名（语言包已整包复用）
6. 用户输入渲染必须过 DOMPurify；不得绕过 request.ts 的加密层
